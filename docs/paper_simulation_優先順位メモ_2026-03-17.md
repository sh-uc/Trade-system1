# paper simulation 観点の優先順位メモ（2026-03-17）

このメモは、`paper_main_2024` の結果をもとに、
`paper simulation` 観点での有力銘柄を整理したものです。

評価軸は次の2つです。

- 利益の大きさ
- 期間分散の良さ

ここでいう期間分散の良さとは、
特定の月や四半期だけに依存せず、複数期間にまたがって取引と利益が出ていることを指します。

### 3.3 総合順位

現時点の `paper simulation` 観点では、次の順が自然。

1. `3103.T`
2. `7013.T`
3. `4506.T`
4. `6235.T`
5. `6330.T`

理由:
- `3103.T` は利益額で最上位。偏りはあるが、期間数も十分ある
- `7013.T` は利益額も大きく、期間分散は最良
- `4506.T` は利益額はやや下がるが、分散が比較的良い
- `6235.T` は利益額は良いが、時期依存がやや強い
- `6330.T` は利益額は良いが、分散面ではやや弱い

このため、`backtest` だけで見ると `6330.T` はかなり強いが、
`paper simulation` の2軸で見ると `7013.T` や `4506.T` の安定感が相対的に上がる。

## 4. 株価上昇率と期間利益の相関

各月・各四半期の株価上昇率と、その期間に発生した `realized_pnl` の相関も確認した。

| ticker | monthly_corr | quarterly_corr |
|---|---:|---:|
| `3103.T` | 0.5608 | 0.9470 |
| `4506.T` | 0.6141 | 0.6603 |
| `6235.T` | 0.7279 | 0.5460 |
| `6330.T` | 0.7506 | 0.9238 |
| `7013.T` | 0.2154 | -0.5604 |

解釈:
- `3103.T`, `6330.T` は、上昇相場の期に利益が出やすい傾向がかなり強い
- `4506.T`, `6235.T` も正の相関があり、上昇局面依存はある
- `7013.T` は相関がかなり弱く、四半期では負になっている

つまり、`7013.T` は
「株価が大きく上がった期にだけ儲かる」型ではなく、
もう少し独自のシグナル発生パターンで利益を積んでいる可能性がある。
ここはむしろ分散面での強みとも読める。
