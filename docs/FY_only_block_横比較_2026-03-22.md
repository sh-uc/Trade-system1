## 6. 型の見え方

今回の横比較からは、次のような仮説が立つ。

- `FY` 前にシグナルが集まりやすい銘柄では、`FY only block` でも影響が大きい
- それ以外の銘柄では、`FY only block` はかなり穏やかな安全策として使える

特に `6235.T` は、外部要因や不安定要素の多さという見立てとも整合しており、
「回避したい気持ちは強いが、利益も削りやすい」銘柄として見える。

---

## 7. 暫定結論

現時点では、`FY only block` は

- 一律全決算 block よりかなり自然
- 多くの銘柄では過剰に効かない
- ただし一部銘柄では十分に利益を削る

という位置づけである。

したがって次の段階では、

1. `FY only block` を基本形として維持する
2. `6235.T` や `3964.T` のように影響が大きい銘柄は個別判断する
3. さらに母集団を広げて、同じ傾向が続くか確認する

の順で進めるのが自然である。
