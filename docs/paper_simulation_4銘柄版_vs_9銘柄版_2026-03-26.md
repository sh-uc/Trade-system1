- 構成としてはかなりきれい

- ただし `FY only block` では `-20,644.80` 減少

## 4. FY only block の影響

### 4銘柄版

- 影響なし
- 今回の期間では、`FY only block` に当たる有効トレードはなかった

### 9銘柄版

- 影響は `3964.T` オークネットの 1 件のみ
- 差分:
  - base: `110,326.45`
  - FY only block: `89,681.65`
  - 差分: `-20,644.80`

したがって、今回の 9銘柄版では
- `FY only block` は全体に雑に効くわけではない
- ただし `3964.T` オークネットでは利益トレードを 1 件止めた
