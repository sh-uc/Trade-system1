### 3.1 有効だった傾向

- `MAX_HOLD_DAYS` は引き続き効きが弱い
- `TAKE_PROFIT_RR` は現在の設計では主役ではなくなりつつある

### 3.3 exit 改良の評価
- 固定 `TP/SL` だけの時点より、`BREAK_EVEN + trailing` の方が明確に良い
- `TRAILING_STOP_R=0.25` が最もきれいで、`0.5` や `0.75` より良かった
