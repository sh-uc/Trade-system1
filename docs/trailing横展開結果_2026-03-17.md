## 4. 分かったこと

今回の上位銘柄では、かなり共通した傾向が出ている。

- `RISK_PCT=0.001,0.0015,0.002` はほぼ差が出ない
- `MAX_HOLD_DAYS=3,5,7` もほぼ差が出ない
- 実質的に効いているのは `MACD_ATR_K`
- `BREAK_EVEN_R=0.5`, `TRAILING_START_R=0.5`, `TRAILING_STOP_R=0.25` は有効な標準条件として使える

つまり、現時点では entry / exit の骨格はある程度定まっており、銘柄ごとの主な差は `MACD_ATR_K` の最適値に表れている。
